Ranges em kotlin ?

ranges são expressoes para uma contagem como ..: de 1 em 1, until: sem contar o ultimo, step: contando com um numero q n seja de 1 em 1, dowTo: é uma contagem regressiva, segue os exemplos abaixo

for(i in 0..3) {             // 1
    print(i)
}
print(" ")

for(i in 0 until 3) {        // 2
    print(i)
}
print(" ")

for(i in 2..8 step 2) {      // 3
    print(i)
}
print(" ")

for (i in 3 downTo 0) {      // 4
    print(i)
}
print(" ")
