oque é herança em kotlin ?

herança é quando uma classe possa herdar suas caracteristicas para outras usando "open", e por sua vez as classes filhas podem sobreescrever usando "override", segue os exemplos abaixo

open class Dog {                // 1
    open fun sayHello() {       // 2
        println("wow wow!")
    }
}

class Yorkshire : Dog() {       // 3
    override fun sayHello() {   // 4
        println("wif wif!")
    }
}

fun main() {
    val dog: Dog = Yorkshire()
    dog.sayHello()
}
