Tipos de dados Kotlin

No Kotlin, o tipo de uma variável é decidido pelo seu valor, segue o exemplo

fun main() {
val myNum = 5                   // Int
    val myDoubleNum = 5.99      // Double
    val myLetter = 'D'          // Char
    val myBoolean = true        // Boolean
    val myText = "Hello"        // String
}
