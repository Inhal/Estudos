oque são classes em kotlin ?

classes são estruturas de dados moldaveis, podendo guardar textos, funções, entre outros para serem usadas futuramentes, segue o exemplo abaixo

class Customer                                  // 1

class Contact(val id: Int, var email: String)   // 2
                        
fun main() {
                        
    val customer = Customer()                   // 3
                            
    val contact = Contact(1, "mary@gmail.com")  // 4
                        
    println(contact.id)                         // 5
    contact.email = "jane@gmail.com"            // 6
}
