oque é for em kotlin ?

for é uma estrutura que realiza uma ação enquanto os criterios ainda não terminaram, ou seja um laço de repetição, segue os exemplos abaixo


fun main(args: Array <String>) {
    val cakes = listOf("carrot", "cheese", "chocolate")

    for (cake in cakes) {                               // 1
        println("Yummy, it's a $cake cake!")
    }

}                   
    


Yummy, it's a carrot cake!
Yummy, it's a cheese cake!
Yummy, it's a chocolate cake!

oque é while e do-while em kotlin ?

o while executa o bloco de codico enquanto a condição é verdadeira e o do-while executa o bloco de codico e depois checa a condição, segue os exemplos abaixo


fun eatACake() = println("Eat a Cake")
fun bakeACake() = println("Bake a Cake")

fun main(args: Array) {
    var cakesEaten = 0
    var cakesBaked = 0
    
    while (cakesEaten < 5) {                    // 1
        eatACake()
        cakesEaten ++
    }
    
    do {                                        // 2
        bakeACake()
        cakesBaked++
    } while (cakesBaked < cakesEaten)

}

oque é interators em kotlin ?

iterators é oque torna uma estrutura de classe ela interavel, segue os exemplos abaixo

class Animal(val name: String)

class Zoo(val animals: List) {

    operator fun iterator(): Iterator {             // 1
        return animals.iterator()                           // 2
    }
}

fun main() {

    val zoo = Zoo(listOf(Animal("zebra"), Animal("lion")))

    for (animal in zoo) {                                   // 3
        println("Watch out, it's a ${animal.name}")
    }

}                   
    


Watch out, it's a zebra
Watch out, it's a lion
