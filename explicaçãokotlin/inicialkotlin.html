O que é Kotlin?

Kotlin é uma linguagem de programação moderna e popular lançada em 2016 pela JetBrains. Tornou-se muito popular porque é compatível com Java (uma das linguagens de programação mais populares que existem), o que significa que o código Java (e bibliotecas) pode ser usado em programas Kotlin. Kotlin é usado para: Aplicativos móveis (especialmente aplicativos Android) desenvolvimento web Aplicativos do lado do servidor Ciência de dados E muito, muito mais!

Por que usar Kotlin?

Kotlin é totalmente compatível com Java Kotlin funciona em diferentes plataformas (Windows, Mac, Linux, Raspberry Pi, etc.) Kotlin é conciso e seguro Kotlin é fácil de aprender, especialmente se você já conhece Java Kotlin é de uso gratuito Grande comunidade/suporte
