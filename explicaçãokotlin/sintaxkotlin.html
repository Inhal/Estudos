Como chamar a função principal ?

a palavra chave "fun" é usada para declarar uma função a função "main()" é a função principal do codigo kotlin, ela é usada para execurtar o codigo dentro das chaves "{}" como por exemplo o "println()" que serve para imprimir uma mensagem, segue o exemplo a baixo 

        fun main() { 
            println("Hello World!")
            println("I am learning Kotlin.")
            println("It is awesome!")
        }
        
        

        Hello World!
        I am learning Kotlin.
        It is awesome!

Como funciona as variaveis ?

as variaveis podem ser atribuidas pelo "var" (variaveis mutaveis) e pelo "val" (variaveis imutaveis) atribuindo um valor com o sinal de igual "=", segue o exemplo a baixo 

        fun main() {
            var name = "John"
            val idade = "25"
            println(name)
            name = "Robert"
            println(name)
            println(idade)
        }
        
        

        John
        Robert
        25
