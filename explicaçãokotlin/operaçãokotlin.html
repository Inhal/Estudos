Como fazer uma soma ?

a função de soma no kotlin é feita com o sinal de adição matematico "+" com um valor numerico ou com uma variavel, segue o exemplo abaixo

fun main() { 
    println( 3 + 3 )
    var x= 5
    var y = 15
    println( x + y )
}

Como fazer um calculo de modulo ?

a função de modulo no kotlin é feita com o sinal de porcentagem "%" que retorna a sobra de uma divisão com um valor numerico ou com uma variavel, segue o exemplo abaixo

fun main() { 
    println( 3 % 2 )
    var x= 19
    var y = 4
    println( x + y )
}

3

Como fazer incremento e decremento ?

a função de modulo no kotlin de incremento é feita com dois sinais de soma matematico "++" e o de decremento é dois de subtração "--" que retorna o proximo valor ou o valor anterior de um valor numerico ou uma variavel, segue o exemplo abaixo

fun main() { 
    println( 3 ++ )
    var x= 19
    println( x-- )
}
        
        

4   
19
